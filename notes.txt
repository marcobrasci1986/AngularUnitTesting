Video instruction to get started: http://blog.jetbrains.com/webstorm/2013/10/running-javascript-tests-with-karma-in-webstorm-7/

Installation:
-----------------
npm install --save-dev karma // install karma
npm install -g karma-cli // install cli
npm install karma-jasmine karma-chrome-launcher --save-dev //  run karma by default with Chrome
npm install --save-dev karma-phantomjs-launcher

Setup karma:
-----------------
karma init

Jasmine: Javascript framework (not only AngularJs)
Karma: Run tests in cli (supports Jasmine)
ngMock: Makes angular testable



Javascript -> Libraries -> manage scopes
